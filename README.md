# Data Warehouse & Modeling

## Overview  
This project implements an end-to-end data warehouse using PostgreSQL, covering data normalization, dimensional modeling, and incremental data processing to support analytical use cases.

## Structure  
- `starter_code/` – Initial schemas, sample data, and partially completed SQL scripts  
- `answer/` – Completed solution with final data models, ETL scripts, and analytics queries  

## Key Components  
- Data normalization in third normal form to ensure data integrity  
- Star schema design with fact and dimension tables for analytics  
- Incremental ETL pipelines for efficient data updates  
- SQL-based analytics using window functions  

## Usage  
1. Run scripts in `starter_code/` to explore the base setup  
2. Refer to `answer/` for the complete implementation  

## Tech Stack  
PostgreSQL, SQL
